Remaining Useful Life Prediction of Aircraft Engines Using Machine Learning

Industrial AI | RNN, LSTM, SVR, CNN, XGBoost | Python, MATLAB

Project Overview

This research focused on developing machine learning models to predict the Remaining Useful Life (RUL) of aircraft engines. The study used run-to-failure data from 260 simulated engines operating across six flight regimes, aiming to predict how many cycles remain before an engine reaches end of life. Accurate RUL prediction plays a vital role in predictive maintenance, cost reduction, and operational safety in aviation systems.

Data & Preparation

  • Dataset included 260 historical engine units for training and 100 units for testing
  • Each unit experienced six operating regimes characterized by altitude, speed, and power settings
  • Sensor selection performed to identify signals strongly correlated with degradation (e.g., pressure ratio, temperature)
  • Data preprocessing included regime-based normalization, batching, and labeling with computed RUL values

Machine Learning Models Used

Recurrent Neural Network (RNN)
  • Implemented in MATLAB using the Levenberg–Marquardt method
  • Trained on raw, unprepared data as a reference baseline
  • Performed well with final RMSE of 34.22
Long Short-Term Memory (LSTM)
  • Developed in Python with tuned memory cells and gate configurations
  • Outperformed all other models with lowest RMSE of 28.23 (mean prediction)
  • Showed robust performance even in noisy early-life data
Support Vector Regression (SVR)
  • Grid search and 10-fold cross-validation used for optimal hyperparameter tuning
  • Offered competitive performance with RMSE of 28.66
1-D Convolutional Neural Network (1D-CNN)
  • Model designed to extract temporal features from engine sensor sequences
  • Achieved RMSE of 32.16 with good predictions near end-of-life regions
Extreme Gradient Boosting (XGBoost)
  • Implemented with tree-based depth control and regularization
  • Achieved average performance with highest RMSE of 37.58 (final prediction)

Results & Model Comparison

All models were evaluated using both their final prediction and an average of their last five predictions. The LSTM model achieved the best performance overall. SVR and 1D-CNN models showed strong predictions for low RUL regions, which is critical for timely maintenance actions. The RNN model, despite using raw data, delivered reasonable accuracy, serving as a benchmark.

Conclusions & Future Work

  • LSTM and SVR showed highest reliability in predicting end-of-life conditions
  • Data preprocessing significantly improved model performance, particularly for SVR and CNN
  • Future work includes sensitivity analysis on data preparation methods and more rigorous grid search across all models
  • Exploring ensemble methods and early-cycle noise reduction may further reduce RMSE
